笔记内容

Redis是一种远程字典服务器，一种基于内存的数据库。Redis通常将数据全部存储在内存上，间隔指定时间也会自动写到硬盘上，但不会在硬盘上取数据，写到硬盘上是为了下次打开Redis时保证数据都还在。运行起来还是会把硬盘上的数据写到内存上使用。

Redis不同于普通的数据库，他就是一个存在内存上的HashMap，数据类型不会有MySQL那么丰富。所以其使用方法也不会有MySQL复杂。其在项目中一般充当如下角色：

  1. 缓存：作为缓存数据库，将常用的数据、计算结果等存储到Redis中，以提高程序性能和响应速度。
  2. 消息队列：通过Redis的发布/订阅机制或者列表结构，实现消息的异步传输和处理，降低系统间耦合度。
  3. 计数器：使用Redis的计数器功能，可以方便地统计网站流量、点击率、在线用户数等数据。
