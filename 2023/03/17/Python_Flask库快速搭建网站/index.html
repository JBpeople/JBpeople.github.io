网络请求

  • get 请求数据会在url中体现
  • post 请求数据不会在url中而是在请求体中

比如在网页登录账号，不能直接通过访问指定网站来实现账号登录，需要提交表单请求。

案例

from flask import Flask, render_template, request

app = Flask(__name__)

# 创建了子网页并将index函数放在该网页下
# 用户一旦访问该网页程序自动执行index函数
@app.route("/show/info")
def index():
    return render_template("web.html")

@app.route("/register")
def register():
    return render_template("register.html")

@app.route("/get/finish")
def get_finish():
    print(request.args)
    return "注册成功"

# 实现POST请求需要备注方法，否则会报错
@app.route("/post/finish", methods=["POST"])
def post_finish():
    print(request.form)
    acc = request.form.get("uu")
    psw = request.form.get("pp")
    gender = request.form.get("gender")
    hobby = request.form.getlist("gender")
    return "注册成功"

# 创建一个即可接收GET也可以接受POST的方法
@app.route("/finish", methods=["GET", "POST"])
def finish():
    if request.method == "GET":
        return "注册成功"
    else:
        print(request.form)
        acc = request.form.get("uu")
        psw = request.form.get("pp")
        gender = request.form.get("gender")
        hobby = request.form.getlist("gender")
        return "注册成功"

if __name__ == "__main__":
    app.run()
