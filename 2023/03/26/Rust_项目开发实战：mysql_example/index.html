笔记内容

书籍中使用的PostgreSQL数据库，为了方便（懒得装）我使用的是MariaDB，该数据库是MySQL的一个分支，由MySQL的原始开发人员创建和维护。MariaDB具有与MySQL类似的功能和语法，因此可以轻松地将现有的MySQL应用程序迁移到MariaDB中。相比于MySQL，MariaDB提供了更多的存储引擎选项、更好的性能和安全性，并且拥有更加积极的社区开发和支持。MariaDB是免费提供给个人和商业用户使用的。

MySQL数据库和前面的SQLite数据库最大的区别就是，MySQL数据库是布置在服务器上的，不再是单独的一个db文件。对在服务器上的数据库进行建表，插值，查询是经常使用的功能。

query_drop() 方法用于执行没有返回结果集的 SQL 语句，例如创建表、删除表等，它会执行 SQL 语句并立即释放资源。该方法可以返回一个 Result<()> 类型的结果，其中 () 表示空元组，表示操作成功完成但没有返回有意义的值。

exec_batch() 方法则用于执行一组带参数的 SQL 语句，例如插入多条记录到数据库中。该方法可以返回一个 Result<BatchResults> 类型的结果，其中 BatchResults 表示一组执行结果，包括每个 SQL 语句的执行状态和受影响的行数等。
